# Kedalup Energy Website

A modern, responsive, and user-friendly frontend web application built using **React.js** and **Vanilla CSS**. This project was developed as a frontend assignment to recreate the Kedalup Energy website design.

## 🚀 Features
- **Responsive Navigation Bar**: Adaptable to Desktop, Tablet, and Mobile viewport sizes.
- **Hero Header**: Engaging full-screen solar farm background with high-impact overlay text and calls-to-action.
- **Challenge & Model Section**: Alternating visual grid showcasing value propositions (Free Installation, Discounted Energy, Full Maintenance, Profit Sharing) alongside the challenge statement and rooftop solar array.
- **Beneficiaries Section**: Multi-column responsive segment outlining the target groups receiving energy support.
- **Enquiry Contact Form**: Interactive input collection with inline field validation (regex matches for phone numbers and email formats) and submission success panels.
- **Professional Footer**: Fully formatted company, quick links, contacts, and copyright rows.

## 📦 Tech Stack
- **Framework**: React.js
- **Tooling**: react-scripts (Create React App standard)
- **Styling**: Vanilla CSS (custom variables, variables-based themes, keyframe animations)
- **Icons**: Lucide React (vector SVG)

## 🛠️ Setup Instructions

Follow these steps to run the project locally on your machine:

### Prerequisite
Make sure you have [Node.js](https://nodejs.org/) installed (version 18 or above recommended).

### 1. Extract and Navigate
Open your command prompt or terminal in the project directory:
```bash
cd "The Kedalup Project"
```

### 2. Install Dependencies
Install all the required React packages and developer tools:
```bash
npm install
```

### 3. Run Development Server
Start the local server. You can use either of these commands:
```bash
npm run dev
# OR
npm start
```
Once started, open [http://localhost:3000](http://localhost:3000) in your web browser.

### 4. Build for Production
To bundle and optimize the project for production deployment, run:
```bash
npm run build
```
This generates a production-ready `build/` folder that can be easily deployed to hosting services like Netlify or Vercel.
